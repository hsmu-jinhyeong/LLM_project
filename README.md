# 프로젝트 제안서

## 프로젝트 개요
- 프로젝트 제목 : 메뉴 추천 챗봇 만들기
- 구성원 : 양인규, 김진형, 한영재, 김규민
- 작성날짜 : 2025/10/13

## 서론 및 배경
매 끼니 뭐 먹을지 고민하는 사람이 많아 인터넷에 뭐 먹을지 추천받는 사람이 증가하는 시대이다. 본 프로젝트는 사용자가 원하는 메뉴를 추천해주는 챗봇을 개발하는 것을 목표로 합니다. 이를 통해 사용자 맞춤형 식사 선택을 돕고, LLM과 조건 기반 필터링을 활용한 실습 경험을 제공합니다.

## 기여점 
간단한 챗봇이지만 조건 기반 필터링을 추가하였고, 데이터셋도 직접 수집하여 정제 경험도 얻음


## 주요 내용
- 구체적인 목표 
- 접근법 포함
- 아키텍처
- 입/출력 인터페이스 등
  
## 자원
- 데이터셋 : 메뉴 종류별(한식, 중식, 일식등) 가짓수 100개 이상

- 개발 도구: Python, OpenAI API or Hugging Face Transformers, Streamlit/Gradio
